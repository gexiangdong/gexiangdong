制作单个文件并可执行的jar

发表于 2018-08-28

通过maven可以把依赖的各种jar和自己的程序都打包成一个jar，这样在部署的时候就非常方便了，这点在做可执行的jar包时就更方便了。
这依赖一个maven的插件：shade

在pom里配置shade插件即可：

使用 mvn package 打包时，就可以把所有依赖的jar都打包到一个jar里了。要运行只需要 java -jar xxx.jar 不用在classpath参数里写一大堆jar了，是不是很方便？

shade插件还可以配置排除/包含某些jar包。
